To compile the code of the VM, do the following:
1. Open Terminal *if terminal is not open already
2. Navigate the directory to file location using "cd"
3. compile using "gcc [file name]"

Before running the code, ensure that the text file that
will be read is within the same folder.
Afterward, run by typing "./a.out [text file name]" within the terminal.
